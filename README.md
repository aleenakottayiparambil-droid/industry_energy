# ⚡ Industrial Energy Optimization AI System

## 🚀 Overview
This project is an AI-powered Industrial Energy Optimization system that predicts energy consumption and carbon emissions based on machine parameters. It also includes an AI chatbot (Gemini) to assist users with insights and recommendations for improving industrial efficiency.

The system helps industries reduce energy waste, optimize machine usage, and support sustainable production.

---

## 🎯 Key Features

- ⚡ Predicts energy consumption based on industrial parameters
- 🌍 Estimates carbon emissions
- 🤖 AI chatbot powered by Google Gemini
- 📊 Smart recommendations for energy optimization
- 🧠 Real-time input-based prediction system
- 🌐 Web-based Flask application

---

## 🧠 Problem Statement
Industries consume large amounts of energy, leading to high costs and carbon emissions. There is a need for an intelligent system that can analyze machine parameters and suggest energy-saving strategies.

This project solves that problem using AI and predictive modeling.

---

## 🏗️ Tech Stack

- Python (Flask)
- HTML, CSS, JavaScript
- Google Gemini AI API
- Flask-CORS
